# 导航与入口

## 桌面端主导航

操作说明：

1. 左侧主导航可在 `分析台 / 问股 / 持仓 / 回测 / 设置` 之间切换。
2. 点击菜单后，右侧内容区会切换到对应功能页。

![桌面端主导航](/img/dsa-ui-manual/01-shell-desktop-navigation.png)

## 移动端导航

操作说明：

1. 移动端底部导航用于快速切换模块。
2. 分析台页面提供“历史/报告”双视图切换。

![移动端主导航](/img/dsa-ui-manual/17-mobile-navigation.png)
