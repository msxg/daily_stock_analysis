# 回测

## 过滤与运行

操作说明：

1. 输入股票代码（可留空全量）。
2. 设置评估窗口并点击“应用筛选”。
3. 点击“运行回测”执行任务。

![回测控制区](/img/dsa-ui-manual/12-backtest-controls.png)

## 结果阅读

操作说明：

1. 查看运行结果摘要（processed/saved/completed 等）。
2. 关注 KPI（如方向准确率）。
3. 结合趋势图与结果表格进行复盘。

![回测结果与 KPI](/img/dsa-ui-manual/13-backtest-results-and-kpi.png)
