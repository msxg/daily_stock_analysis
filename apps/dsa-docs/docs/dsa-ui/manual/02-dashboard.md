# 分析台

## 提交分析任务

操作说明：

1. 在“股票代码/名称”输入框输入标的。
2. 根据需要勾选或取消“推送通知”。
3. 点击“提交分析”，观察结果提示。

![提交分析任务](/img/dsa-ui-manual/02-dashboard-submit-analysis.png)

## 查看任务、历史与报告快捷操作

操作说明：

1. 左侧查看任务状态与历史分析。
2. 在报告区使用 `复制摘要 / 查看 Markdown / 追问 AI` 快捷操作。

![历史与报告操作](/img/dsa-ui-manual/03-dashboard-history-and-actions.png)

## 查看 Markdown 报告

操作说明：

1. 点击报告标签中的 `Markdown`。
2. 在 Markdown 内容区查看原始报告文本。

![Markdown 报告](/img/dsa-ui-manual/04-dashboard-markdown-tab.png)
