# 持仓

## 顶部控制与禁写保护

操作说明：

1. 先选择账户和成本法。
2. 当账户为“全部账户”时，系统会显示禁写提示。
3. 需要写入（录入/导入/删除）时，先切换到具体账户。

![持仓顶部控制与禁写](/img/dsa-ui-manual/08-portfolio-overview-and-protection.png)

## 手工录入交易

操作说明：

1. 切换到“手工录入”标签。
2. 输入股票代码、数量、价格等字段。
3. 点击“提交交易流水”完成录入。

![手工录入交易](/img/dsa-ui-manual/09-portfolio-entry-form.png)

## CSV 导入

操作说明：

1. 切换到“CSV 导入”标签。
2. 选择券商格式并上传 CSV 文件。
3. 点击“解析文件”预览后，再点击“提交导入”。

![CSV 导入流程](/img/dsa-ui-manual/10-portfolio-import-flow.png)

## 流水与修正

操作说明：

1. 切换到“流水与修正”标签。
2. 选择流水类型并刷新。
3. 在流水列表中进行核对与后续修正操作。

![流水与修正](/img/dsa-ui-manual/11-portfolio-events-management.png)
