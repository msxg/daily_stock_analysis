# Daily Stock Analysis 文档中心

这里是 `apps/dsa-docs` 的 Docusaurus 文档站点，用于承载 `dsa-ui` 重构交付文档与操作手册。

## 快速入口

- [DSA UI 操作手册](./dsa-ui/user-manual)
- [DSA UI 重构方案](./dsa-ui/redesign-proposal)

1. 新成员先看“重构方案”了解目标和范围。

3. 业务和测试成员按照“操作手册”执行功能验收。
